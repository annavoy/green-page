# Green Harvest Landing Page

Single-page responsive landing website for a vegetable delivery service.

### Features
- Responsive layout for mobile, tablet, and desktop
- Burger menu for small screens
- Smooth section navigation with anchor links
- Product cards, reviews, and order form sections
- Scroll animations via AOS

### Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

### Project Structure
- `index.html` — page markup
- `css/style.css` — all styles and responsive breakpoints
- `js/main.js` — mobile menu logic
- `img/` — images and SVG sprite

### Run Locally
1. Clone the repository.
2. Open the project folder.
3. Run with Live Server (recommended) or open `index.html` in your browser.
